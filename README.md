# vue-markdown
基于vue2，vuex，element-ui，highlight.js，markdown-it的markdown编辑器

### 如何启动
进入根目录下，执行如下命令

npm install

npm run serve

### 打包
npm run build

### 运行在你的服务器上
npm install -g serve

serve -s build

### 支持的功能
- 工具栏
  - 下载你编辑的md文档（手机端浏览器不支持）
  - 一键清空所有内容
  - 加粗、斜体、下划线、代码块、注释、插入图片、插入超链接
- 快捷键
  - Ctrl+Z 撤回3s之前的文本
  - Ctrl+B 加粗
  - Ctrl+I 斜体
  - ctrl+shift+k 代码块
  - ctrl+shift+i插入图片
